## What is Dropmute?

A lightweight, native macOS app that converts files instantly. Drop a file, pick a format, done. No uploads, no cloud, no subscriptions — everything runs locally on your Mac.

## Supported Formats

| Images | Documents |
|---|---|
| PNG, JPEG, WebP, HEIC, HEIF, AVIF, TIFF, BMP, GIF, SVG, ICO | PDF, DOCX, Markdown, HTML, CSV, JSON, EPUB, RTF, TXT, XLSX |

## Features

- **Drag & drop** — drop files, folders, or paste from clipboard (Cmd+V)
- **Batch conversion** — drop a folder, pick formats per file type, convert all at once
- **Format guide** — every format explained with best-for tags so you always pick the right one
- **Favorites** — right-click any format to pin it for quick access
- **Smart memory** — remembers your last-used format per input type
- **Live size estimate** — see the output size before converting, adjusts with quality slider
- **Per-conversion output folder** — save anywhere, override the default with one click
- **Conversion history** — review past conversions, reveal in Finder
- **Keyboard shortcuts** — Enter to convert, Esc to clear
- **Menu bar icon** — always accessible, even when the window is closed
- **Auto-updates** — via Sparkle, never manually download again
- **macOS 26 Liquid Glass** — native look on Tahoe, graceful fallback on older macOS
